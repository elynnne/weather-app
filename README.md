# Weather App 🌤️

A modern weather web app built with Python and Flask that shows real-time weather data and 5-day forecasts with a clean dark UI.

## Features
- Real-time weather data for any city
- Shows temperature, feels like, humidity and wind speed
- 5-day forecast
- Modern dark UI design

## How to run
1. Install dependencies: pip install flask requests
2. Get a free API key from openweathermap.org
3. Add your API key to main.py
4. Run: python main.py
5. Open http://127.0.0.1:5000 in your browser

## Technologies
- Python
- Flask
- OpenWeatherMap API
- HTML & CSS
