# Weather App
A simple Python app that gets real time weather data by using the OpenWeatherMap API.

## Features
- Get current weather for any city
- Shows temperature, feels like, humidity and description

## How to run
1. Install requests: pip install requests
2. Get a free API key from openweathermap.org
3. Add your API key to main.py
4. Run: python main.py

## Technologies
- Python
- OpenWeatherMap API
